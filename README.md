# ContabilometriaRevisao - Dataset para Aplicação de Correlação e Regressão

Este repositório contém o dataset `ContabilometriaRevisao` utilizado para a revisão da prova de Contabilometria, focado na aplicação de métodos de correlação e regressão linear.

## Descrição do Dataset

O dataset é composto por **70 observações**, cobrindo um período de **7 anos (2019-2025)** para um conjunto de empresas.

## Colunas (Variáveis)

O arquivo `ContabilometriaRevisao.csv` possui as seguintes variáveis:

ID_Empresa -	Identificador único da empresa -	Qualitativo (ID)
Ano -	Ano de observação (2019 a 2025) -	Quantitativo (Discreto)
Setor -	Setor de atuação da empresa -	Qualitativo (Categórico)
Ativo_Total -	Valor total dos ativos da empresa -	Quantitativo
Receita_Vendas -	Receita total de vendas -	Quantitativo
Lucro_Liquido -	Lucro líquido no período -	Quantitativo
Liquidez_Corrente -	Índice de liquidez corrente (ativo circulante / passivo circulante) -	Quantitativo (Índice)
ROA -	Return on Assets (Retorno sobre Ativos) -	Quantitativo (Índice/%)
Endividamento -	Nível de endividamento da empresa -	Quantitativo (Índice/%)
Inadimplencia -	Inadimplência observada - Dummy (0/1)
